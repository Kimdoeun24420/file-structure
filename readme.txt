학생 및 수강 등록 파일 입출력 예제

본 C++ 프로그램은 학생 및 수강 등록 데이터를 관리하기 위한 파일 입출력 작업을 보여줍니다. 이 프로그램은 Student와 CourseRegistration 두 개의 클래스를 정의하며, 각 클래스에는 파일로 데이터를 패킹하고 언패킹하는 메서드가 포함되어 있습니다.

준비물

C++ 컴파일러
표준 템플릿 라이브러리(STL)

사용 방법

이 저장소를 로컬 머신에 복제합니다:
[git clone <repository_url>]

소스 코드가 있는 디렉토리로 이동합니다:
[cd <repository_directory>]

C++ 컴파일러를 사용하여 프로그램을 컴파일합니다. 예를 들어, g++을 사용할 경우:
[g++ main.cpp -o main]

컴파일된 실행 파일을 실행합니다:
[./main]


기능
Student 클래스: 학생 ID, 이름, 주소, 첫 수강일 및 학점 시간을 포함한 학생 정보를 나타냅니다. 이 클래스는 데이터를 파일로 패킹하고 언패킹하는 메서드를 제공합니다.
CourseRegistration 클래스: 과목 등록 정보를 나타냅니다. 과목 ID, 학생 ID, 학점 시간 및 성적을 포함합니다. Student 클래스와 마찬가지로 데이터를 파일로 패킹하고 언패킹하는 메서드를 제공합니다.
파일 입출력 작업: 학생 및 수강 등록 데이터를 저장하고 검색하는 파일 입출력 작업을 보여줍니다.

파일 구조
main.cpp: Student 및 CourseRegistration 클래스 사용법을 보여주는 메인 프로그램 코드를 포함합니다.
